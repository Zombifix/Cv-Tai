# Agent Bootstrap

Ce repo utilise `CLAUDE.md` comme mémoire canonique.

## À lire en premier
1. `CLAUDE.md`
2. `.ai/TODO.md`
3. `.ai/LAST_CHANGES.md`
4. `git status --short`
5. `git diff --name-only`

## Règles
- Ne pas refaire une review complète du repo sauf demande explicite.
- Commencer par les fichiers du diff et les fichiers listés dans `.ai/TODO.md`.
- Mettre à jour `.ai/TODO.md` et `.ai/LAST_CHANGES.md` après tout changement significatif.
